이소연의 웹사이트

강의 계획서

안녕하세요? 저는 메이킷코드랩 금요일 15시 웹 프로그래밍 수업을 담당하고 있는 이소연 연구원이라고 합니다! 웹 프로그래밍 수업을 듣는 학생들은 현재 영역에 자기 소개를 넣거나 Javascript로 간단한 미니 게임을 만들어 넣고 싶다고 했습니다. 다음 시간에는 간단한 로그인 기능을 구현하기 위해 Javascript 기본 문법 중 변수 선언, 조건문, 함수를 우선적으로 파이썬과 비교하며 배우고, HTML 태그를 선택하기 위한 DOM 객체를 다루는 방법을 학습할 계획입니다.

먼저 login.html을 새로 만들고 아직 배우지 않은 form 태그, input 태그, button 태그를 사용하여 심플한 로그인 폼을 만들려고 합니다. 그리고 CSS로 HTML 요소를 숨길 수 있는 display 속성을 배울 계획입니다. document 객체나 querySelector 함수를 어떻게 하면 쉽게 설명할 수 있을지 조금 걱정이 되기도 합니다. 초등학생인 학생들 수준에 맞게 과도하게 어려운 부분은 간단하게만 알려주고 재미있게 실습을 진행할 수 있도록 고민하겠습니다!
